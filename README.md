Visualing Cancer Drug Discovery patent metadata: For this project, I chose an open data set provided by the USPTO that consisted of metadata information for cancer research patent documents. This open data set was created by the U.S. Patent and Trademark office to assist cancer researchers in understanding what research was being performed by other cancer researchers. The data set included R&D information in diagnostics, therapeutics, data analytics, and model biological systems. The data set is an excel file in csv format. There are twenty-one columns: The first twelve columns included text/numeric data values and the remaining nine columns were Boolean (0 or 1) data values.  The data set and accompanying document explaining the data set are located here: https://developer.uspto.gov/product/cancer-moonshot-data-ms-excel-csv#product-files ...
